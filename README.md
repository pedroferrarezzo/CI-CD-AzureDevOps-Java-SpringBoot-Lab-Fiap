# Simple API Java

API Java Simples para lab de integração com o AzureDevOps Board

## Pré-requisitos

- Java 21
- Git
- Docker

## Build e execução

```sh
docker compose up --build
```

## Testes unitários (validação) - Maven

./mvnw test


## Documentação online (OpenAPI)
